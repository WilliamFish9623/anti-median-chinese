# 词汇与句式替代清单

本文件供 SKILL.md 第二步「改造」加载使用。提供具体的替代选项，但不强制套用。

**核心原则**：替代不是堆砌。每段最多用 2 到 3 个非高频替代词，多了就变成另一种刻意。

**风格标签**：

- **口语**：适合日常表达、公众号评论、个人经验分享
- **书面**：适合评论、散文、较正式的文章
- **半书面**：介于口语和书面之间，适合有一点文气但不装腔的文本
- **慎用**：容易显得地域化、年代感强或用力过猛，只有贴合作者声音时才用

## 二、程度词替代

### 高（替代「非常」「很」）

| 通用 | 替代选项 |
|---|---|
| 非常 + adj | 极（书面）、颇（半书面）、着实（书面）、煞是（半书面，慎用）、委实（书面，慎用）、十分（书面）、相当（通用） |
| 非常 + adj | 真（口语）、太（口语）、超（年轻化口语）、贼（北方口语，慎用）、老（北方口语，慎用） |
| 非常重要 | 关键（通用）、要紧（口语）、紧要（书面）、至关重要（正式）、举足轻重（书面，慎用） |
| 非常好 | 极好（书面）、妙（半书面）、漂亮（口语/评论）、顶好（口语，慎用）、绝（年轻化口语）、过瘾（口语） |
| 非常喜欢 | 偏爱（书面）、中意（口语/地域，慎用）、钟爱（书面）、痴迷（书面）、上头（年轻化口语） |
| 非常困难 | 棘手（通用）、麻烦（口语）、不好办（口语）、难办（通用）、烫手（口语/比喻） |
| 非常容易 | 不在话下（半书面，慎用）、随手（口语）、信手（书面）、轻松拿下（口语） |

### 中（替代「比较」「还算」）

| 通用 | 替代 |
|---|---|
| 比较 + adj | 略（书面）、稍（通用）、还算（口语）、算得上（通用）、约莫（口语/地域，慎用） |
| 有点 + adj | 略带（书面）、稍显（书面）、不无（书面，慎用）、多少有点（口语） |

### 选用要点

- 偏口语场景：「真」「太」「贼」「老」「死」（如「死贵」）
- 偏书面场景：「极」「颇」「着实」「委实」
- 不要混搭：一段里口语和书面的程度词不要乱跳
- 自评测试：作者自己日常会说哪个？用那个

## 三、虚词与语气助词

### 句末语气助词

| 助词 | 语气 | 示例 |
|---|---|---|
| 呢 | 口语；反问、强调、保留 | 「那又怎样呢？」 |
| 啊 | 口语；感叹、强调、亲近 | 「这事儿啊，难办。」 |
| 吧 | 口语；推测、商量、让步 | 「就这样吧。」 |
| 呗 | 口语；随意、无奈、轻松，慎用 | 「不来就不来呗。」 |
| 嘛 | 口语；解释、撒娇、不耐，慎用 | 「我说了嘛，不行。」 |
| 啦 | 口语；完成、感叹、轻松，慎用 | 「都搞定啦。」 |
| 哦 | 口语；提醒、轻嘱、应答，慎用 | 「记得带钥匙哦。」 |
| 罢了 | 书面；收尾、轻描淡写，慎用 | 「不过尔尔罢了。」 |

### 副词层次

| 单调版 | 有层次版 |
|---|---|
| 也 / 还 / 再 | 还是（通用）、倒也（半书面）、依旧（书面）、也罢（书面，慎用）、便也（书面，慎用） |
| 但是 / 不过 | 却（通用）、倒是（口语/半书面）、反倒（通用）、偏偏（口语/叙事）、可（口语） |
| 因为 | 毕竟（通用）、说到底（口语）、由于（书面）、缘于（书面，慎用） |
| 所以 | 就（口语）、于是（通用）、便（书面，慎用）、故而（书面，慎用） |

### 口语态度短语

适合在段首或转折处用，制造态度：

- 「说白了」「话说回来」「再说了」「你看」
- 「我跟你讲」「老实说」「不瞒你说」
- 「换个说法」「换句话讲」
- 「这么说吧」「这么讲吧」

## 四、句式调整

### 拆长成短

**长句模式**（AI 倾向）：
> 因为这个方案在执行过程中遇到了来自多方面的阻力，所以最终的结果并不理想，需要重新调整。

**拆短模式**：
> 这个方案执行不下去。阻力太多，结果不理想，得重调。

### 单句成段（强调技巧）

把一个关键判断从段落里拎出来，单独成段。这是最强的强调手法。

**埋没版**：
> 我试了三个方案都失败了，最后发现根本问题是我从一开始就误解了需求，需要回到原点重来。

**强调版**：
> 我试了三个方案都失败了。
>
> 最后发现，根本问题是我一开始就误解了需求。
>
> 得回到原点重来。

### 反问替代陈述

强化态度时用反问，比陈述句有力。

| 陈述 | 反问 |
|---|---|
| 这件事不划算 | 这件事划得来吗？ |
| 这个方案行不通 | 这方案能行吗？ |
| 我们应该尝试 | 不试一下怎么知道？ |
| 这不是问题 | 这算什么问题？ |

注意：反问句一段最多用 1 次，过密就变成质问。

## 五、留白手法

### 主语省略

只在主语切换或开头出现，重复出现时省掉。

**过满**：
> 我早上六点起床。我洗漱完出门。我坐地铁去公司。我九点准时打卡。

**留白**：
> 早上六点起床，洗漱出门，地铁到公司，九点准时打卡。

### 关系省略

不一定每个「因为」都要「所以」，逻辑可以隐含。

**过满**：
> 因为今天下雨了，所以我没去公园。

**留白**：
> 下雨了，没去公园。

### 时间副词省略

「然后」「接着」「之后」频繁出现是 AI 味的强信号。能删则删。

**过满**：
> 我先做了 A。然后我做了 B。接着我做了 C。最后我做完了 D。

**留白**：
> A、B、C 一路做下来，D 收尾。

## 六、不要用的「反 AI 味」陷阱

以下手法看起来「有质感」，实际是另一种中位数。慎用：

- **生硬堆砌成语**：一段里出现 3 个以上成语
- **强插古诗词**：没有上下文铺垫的引用，显得拽
- **滥用文言虚词**：「之」「乎」「者」「也」突然冒出来
- **过度比喻**：每句话都要打比方
- **「仿汪曾祺」「仿王小波」**：模仿某个作家的标志性句式
- **四字短语堆砌**：「秋高气爽，月明星稀，万籁俱寂」式的小红书古早文艺
- **过度短句**：所有句子都是 5 个字以内，变成口号

判断标准：让作者自己读改造后的版本，他会觉得「这是我写的」还是「这是装出来的」？后者就回退。
