# anti-median-chinese

一个用于诊断并改善「中位数中文」的 Codex skill。「中位数中文」指 AI 时代常见的扁平中文：技术上通顺，表达上寡淡；能把信息说清楚，却缺少画面、节奏、语气和人的痕迹。

这个 skill 不负责从零写文章，而是用于稿件完成后的质感打磨：

- 识别动词上位化、程度词同质化、语气缺失、句长均匀、留白消失、标点单调等症状
- 给出具体、可逆、不过度文饰的改造建议
- 提供局部示范，除非用户明确要求，不默认全文重写

## 文件结构

- `SKILL.md`：skill 主说明和工作流程
- `references/diagnostics.md`：诊断清单
- `references/examples.md`：诊断与改造案例
- `references/word-bank.md`：词汇、虚词和句式参考

## 使用场景

适合：

- 去掉 AI 味
- 给公众号文章终稿加一点质感
- 诊断中文为什么「通顺但没味道」
- 调整画面感、节奏和个人声音

不适合：

- 技术文档
- API 说明
- 合规文本
- 客服话术
- 明确要求简洁、严肃、公文体的文本

核心判断是：这段中文的目标是「传递信息」，还是「传递人」。
